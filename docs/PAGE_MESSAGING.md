# ISJ v4 — Page Messaging Guide

Brief, guiding. Not a script. Each page should feel like ISJ, not like a template.

Themes that run through the whole site:
- Every child is known by name
- Teachers who came here by choice, from the best schools
- Children learn how to think, not what to think
- Risk-taking, confidence, joy in learning
- British curriculum, Jakarta location — that's the product, say it clearly
- Show, don't list. One vivid detail beats three bullet points.

---
